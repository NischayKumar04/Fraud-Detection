# Data

This folder contains the dataset files. They are **not included in the repository** due to size.

## How to Get the Data

1. Go to [Kaggle IEEE-CIS Fraud Detection](https://www.kaggle.com/c/ieee-fraud-detection/data)
2. Download:
   - `train_transaction.csv`
   - `train_identity.csv`
3. Place both files in this `data/` folder
